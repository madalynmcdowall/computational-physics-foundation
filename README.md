# computational-physics-foundation

Numerical methods and physics simulations in Fortran and C++.
Developed for PHY 277: Computation for Physics and Astronomy.

### Fortran Projects
- **Kinetic Energy Calculator** - Relativistic vs non-relativistic comparison
- **Binary Star Magnitude** - Periodic light curve calculation
- **Series Summation** - Finite series evaluation
- **Simpson's Rule Integration** - Numerical integration with convergence analysis
- **Statistical Data Analysis** - Mean, median, standard deviation from file I/O
- **Poisson Equation Solver** - 2D electrostatic potential with iterative methods
- **Double Precision Simpson's Rule** - Advanced numerical integration with modules

### C++ Projects
- **Quadratic Root Finder** - Real and complex root calculation
- **Newton-Raphson Root Solver** - Iterative root finding with convergence analysis
- **Least Squares Linear Fit** - Data fitting with file I/O and dynamic memory
